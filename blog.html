Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM. They are mentioned below:
1. By ID(document.getElementById('ID Name')),
2. By Class(document.getElementByClassName('class-name')),
3. By Tag(document.getElementbyTagName('Tag-Name')),
4. By QuerySelector(document.querySelector('selector class')),
5. By querySelectorAll(document.querySelectorAll('slector name')),
Note: In query selector, one have to write the selector name as per the css rules.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerText, innerHTML, textContent are properties of Javascript DOM to manipulate what's inside an HTML element. Their purpose of serving are slightly different as mentioned below:
innerText: This only sets or gets the text inside of an element. No html element or tag and hidden text won't be shown in innerText. This is basically used to manipulate only texts inside an element.
innerHTML: This sets or gets both texts and html elements iside an html tags. However, it doesn't include hidden texts inside the element.
textContent: This sets or gets the text content inside an element. It includes texts, html tags, hidden texts. However, in textContent html tags doesn't act like tags, but acts only as plaine texts.

Question-3: What is event delegation in the DOM?

Event delegation is used in Javascript for handling event in DOM. Various techniques are used to handle events to improve performance and event delegatin is works as below: Event Bubbling: Event bubbling in the Document Object Model (DOM) is a mechanism in JavaScript where an event triggered on a child element of a container also gets propagated to the parent elements unless stopped.
Event Listener: Event Listener mainly used to handle a single event in a single element, instead of bubbling inward or outward. It is very helpful to handle eevent in a selected element.
Event Targeting: Event targeting allows us to target in which element the event has happened and manipulate it. Simply, event.target property is used to indicate that element.

Question-4: What is event bubbling in the DOM?

Event bubble refers to the incident where an event triggered in a child element of a container gradually propagates outward to its parents as well as container element respectively.
This allows developers to handle events on multiple levels of the DOM hierarchy with a single event listener.
The direction of the event bubble is from target to root.

Question-5: How do you create, add, and remove elements using JavaScript?

Various methods can be used to create, add and remove elements using Javascript in DOM. Some of the popular methods are as below: 1. To create element: For creating elements createElement() method is used commonly. Code snippet- document.createElement('element name');
2. To add element: For adding elements, mainly appendChild() method is used. Code snippet - parenteElementName.appendChild(createdElementName);
3. To remove element: For removing element, mainly removeChild() method is used. Code snippet - parenteElementName.removeChild(element name that is to be removed);
